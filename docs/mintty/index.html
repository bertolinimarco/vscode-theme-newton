Newton

A low-eyestrain MinTTY Theme

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo directly into your ~/. directory.

$ git clone https://github.com/bertolinimarco/mintty-theme-newton

Install manually

  1. Download
  2. Rename to .minttyrc and Move the file to ~/.

Activating theme

If you're already running a color scheme under Mintty, be sure to clear the relevant lines in your ~/.minttyrc.
