Install using Command Palette

  1. Go to View > Command Palette or press Ctrl+Shift+P
  2. Then enter Install Extension
  3. Write Newton
  4. Select it or press Enter to install

Select Theme

Run Visual Studio Code. The Newton Syntax Theme will be available from File > Preferences > Color Theme dropdown menu.

Install via Studiostyl.es

https://studiostyl.es/schemes/newton

Activating theme

  1. Start Visual Studio
  2. Go to Tools -> Import and Export Settings... -> Import selected environment settings
  3. [Choose whether you want to backup existing settings]
  4. Click Browse and select the .vssettings file
  5. Click Finish


Install using Git

You can install the theme and keep up to date by cloning the repo directly into your Packages directory in the Sublime Text application settings area.

You can locate your Sublime Text Packages directory by using the menu item Preferences > Browse Packages....

While inside the Packages directory, clone the theme repository using the command below:

$ git clone https://github.com/bertolinimarco/sublime-theme-newton

Install manually

  1. Download
  2. Unzip the files and rename the folder to Newton Color Scheme
  3. Find your Packages directory using the menu item Preferences > Browse Packages...
  4. Copy the folder into your Sublime Text Packages directory

Activating theme

Go to Preferences > Color Scheme > User and select the Newton Color Scheme.

Install using Atom

  1. Go to Atom -> Preferences...
  2. Then select the Install tab
  3. Click the Themes button to the right of the search box
  4. Enter newton-syntax in the search box

Install using Git

Alternatively, if you are a git user, you can install the theme and keep up to date by cloning the repo directly into your ~/.atom/packages directory.

$ git clone https://github.com/bertolinimarco/newton-syntax

Install manually

  1. Download
  2. Move the newton-syntax folder to ~/.atom/packages

Activating theme

Go to Atom -> Preferences..., click in the Themes tab, and select Newton in the Syntax Theme dropdown.

Install using hpm

Install via hpm command line:

$ hpm install hyper-theme-newton

Install using config file

Add hyper-theme-newton to the plugins list in your ~/.hyper.js config file.

plugins: [ "hyper-theme-newton" ]

Activating theme

  1. Start Hyper
  2. Go to View > Full Reload
