Newton

A low-eyestrain Hyper Theme

Star

Install using hpm

Install via hpm command line:

$ hpm install hyper-theme-newton

Install using config file

Add hyper-theme-newton to the plugins list in your ~/.hyper.js config file.

plugins: [ "hyper-theme-newton" ]

Activating theme

  1. Start Hyper
  2. Go to View > Full Reload
