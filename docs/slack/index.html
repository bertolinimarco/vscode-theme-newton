A low-eyestrain Slack Theme

Install manually

  1. Go to User Menu > Preferences > Sidebar > Theme
  2. In the bottom of the window, look for the textbox called "Copy and paste these values to share your custom theme with others"
  3. Copy and paste the values:
#1a1a1a,#666666,#ebbd5b,#1a1a1a,#666666,#d3d0c8,#9c9,#f2777a
