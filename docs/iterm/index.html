Newton

A low-eyestrain iTerm Theme

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

$ git clone https://github.com/bertolinimarco/iterm-theme-newton

Install manually

Download

Activating theme

  1. iTerm2 > Preferences > Profiles > Colors Tab
  2. Open the Color Presets... drop-down in the bottom right corner
  3. Select Import... from the list
  4. Select the Newton.itermcolors file
  5. Select Newton from Color Presets...
