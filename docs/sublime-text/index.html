Newton

A low-eyestrain Sublime Text Theme



Install using Git

You can install the theme and keep up to date by cloning the repo directly into your Packages directory in the Sublime Text application settings area.

You can locate your Sublime Text Packages directory by using the menu item Preferences > Browse Packages....

While inside the Packages directory, clone the theme repository using the command below:

$ git clone https://github.com/bertolinimarco/sublime-theme-newton

Install manually

  1. Download
  2. Unzip the files and rename the folder to Newton Color Scheme
  3. Find your Packages directory using the menu item Preferences > Browse Packages...
  4. Copy the folder into your Sublime Text Packages directory

Activating theme

Go to Preferences > Color Scheme > User and select the Newton Color Scheme.
