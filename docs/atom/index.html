Newton

A low-eyestrain Atom Theme

Install using Atom

  1. Go to Atom -> Preferences...
  2. Then select the Install tab
  3. Click the Themes button to the right of the search box
  4. Enter newton-syntax in the search box

Install using Git

Alternatively, if you are a git user, you can install the theme and keep up to date by cloning the repo directly into your ~/.atom/packages directory.

$ git clone https://github.com/bertolinimarco/newton-syntax

Install manually

  1. Download
  2. Move the newton-syntax folder to ~/.atom/packages

Activating theme

Go to Atom -> Preferences..., click in the Themes tab, and select Newton in the Syntax Theme dropdown.
